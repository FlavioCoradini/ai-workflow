# Security Policy

This repo holds Agent Skills and a shell installer that symlinks them into your local AI harness
directories. `install.sh` only creates symlinks under your own home directory and never sends
anything off your machine. Some skills ship helper scripts under `skills/<name>/scripts/` — read a
skill before installing it, the same as any code you run locally.

## Reporting a vulnerability

Please don't open a public issue for a security problem. Use GitHub's
[private vulnerability reporting](https://github.com/flaviocoradini/ai-workflow/security/advisories/new),
or email coradiniflavio@gmail.com. I'll get back to you within a few days.
